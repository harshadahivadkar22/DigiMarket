# DigiMarket 🛒

A web-based E-Commerce Management System developed using Java, Spring MVC, Hibernate, JSP, and MySQL. DigiMarket provides an efficient platform for managing products, customers, orders, and online shopping activities.

## 📌 Project Overview

DigiMarket is designed to simplify online shopping by providing a user-friendly interface for customers and an administration panel for managing products and orders.

## ✨ Features

### 👨‍💼 Admin Module

* Admin Login
* Add, Update, and Delete Products
* Manage Categories
* View Customer Details
* Manage Orders
* Dashboard Analytics

### 👤 Customer Module

* User Registration & Login
* Browse Products
* Search Products
* Add Products to Cart
* Place Orders
* View Order History

### 🛍️ Product Management

* Product Listing
* Product Details
* Category-wise Products
* Stock Management

## 🛠️ Technologies Used

## 🚀 Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/harshadahivadkar22/DigiMarket.git
```

### 2. Import Project

* Open Eclipse
* File → Import
* Existing Maven Project
* Select Project Folder

### 3. Configure Database

Create a MySQL database and update the database credentials in the configuration file.

### 4. Build Project

```bash
mvn clean install
```

### 5. Deploy on Tomcat

* Add project to Tomcat Server
* Start the server

### 6. Run Application

```text
http://localhost:8080/DigiMarket
```

## 📊 Workflow

1. Admin manages products and categories.
2. Customers register and log in.
3. Customers browse products and add them to the cart.
4. Orders are placed and stored in the database.
5. Admin manages orders and inventory.
